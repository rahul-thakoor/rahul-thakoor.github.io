A new version packed with new features and improvements

 Posted on February 27, 2019  |  2 minutes  |  Rahul Thakoor

For the past month I have been working on rust_gpiozero. This new release is a complete rewrite and uses Rene van der Meer(@golemparts)’s [rppal](https://github.com/golemparts/rppal) crate under the hood. It also uses Rust 2018 edition. Why rppal? Simply because Rene has done an incredible work on rppal. It provides first class support for the Raspberry Pis.  Posted on July 7, 2018  |  3 minutes  |  Rahul Thakoor

A couple of months ago I released the rust_gpiozero1 crate. It is a port of the GPIO Zero2 library by the Raspberry Pi Foundation. The library provides a simple interface to control GPIO devices with a Raspberry Pi. Following this, recently ported the Raspberry Pi Foundation’s Physical Computing with Python3 guide for Rust.
